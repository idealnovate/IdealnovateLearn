Programme Overview

2-week live online bootcamp designed for absolute beginners with zero tech or data background.

2Weeks
14hrsLive Sessions
5+Projects
3Tools Mastered

What's included:

  • Live instructor-led sessions
  • Assignments & expert feedback
  • Real-world data project support
  • Global certificate upon completion
  • Definition & importance of data analysis in business and everyday decisions
  • Core data concepts — structured vs unstructured data, data types
  • Types of data analytics: descriptive, diagnostic, predictive, prescriptive
  • Roles in data: analyst, engineer, scientist — what each does and earns
  • Introduction to Microsoft Excel — navigating the interface
  • Creating, saving & exporting files; working with sheets & workbooks
  • Pivot tables & pivot charts — your first look at summarising data
Practical: Basic data visualisation in Excel — building charts & graphs from a real sample dataset.
  • Formatting data — number formats, conditional formatting, cell styles
  • Customising tabs & groups for a faster workflow
  • Sorting & filtering datasets to find insights quickly
  • Critical formulas & functions: SUM, AVERAGE, MIN, MAX, COUNT, COUNTA
  • Arithmetic operators and nested formulas for real calculations
  • TEXT functions — manipulating and cleaning text-based data
  • Introduction to Microsoft Power BI — installation & setup
Assignment: Sales Data Project — analyse a real sales dataset, apply key formulas, and present findings with charts.
  • Data visualisation concepts — why visuals communicate faster than tables
  • Power BI overview: Desktop, Service, and Mobile — what's what
  • Navigating the Power BI interface: report view, data view, model view
  • Loading and transforming data using Power Query
  • Power BI chart types — bar, line, pie, map, card, and matrix visuals
  • Building and formatting your first interactive dashboard
Practical: Build basic Power BI visualisations + create a fully interactive dynamic dashboard from a provided dataset.
  • Opportunities in data analysis — local, remote, and freelance markets
  • How to monetise your data skills: full-time roles, contracts, consulting
  • Job access as a data analyst — building your CV, portfolio & LinkedIn
  • What hiring managers look for in entry-level data analyst candidates
  • Salary expectations across Africa, Europe, and remote global roles
Live AMA: Open Q&A session with Idealnovate Africa industry mentors — bring your career questions.
  • Introduction to databases — what they are and how data is stored
  • Introduction to SQL — why every data analyst needs it
  • SQL commands & operations: CRUD (Create, Read, Update, Delete)
  • Creating SQL tables and defining a SQL database structure
  • Querying data — SELECT, FROM, WHERE, ORDER BY, LIMIT
  • Filtering and aggregating results with GROUP BY and HAVING
Practical: Run basic SQL queries on a real sample database — retrieve, filter, and count records from scratch.
  • Working through a real-world business case study from start to finish
  • Defining analysis objectives and understanding the business question
  • Data cleaning — identifying errors, duplicates, nulls, and inconsistencies
  • Data visualisation & storytelling — presenting findings to stakeholders
  • Building your final project report and dashboard for your portfolio
  • Graduation — certificate issuance and community onboarding
Practical: Hands-on end-to-end capstone analysis — clean, analyse, visualise, and present a real dataset.
